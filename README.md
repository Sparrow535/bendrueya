# Bendrueya Frontend

React frontend scaffolded with Vite.

## Requirements

- Node.js 22+
- npm 10+

## Development

```bash
npm install
npm run dev
```

The development server starts on `http://localhost:5173` by default.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build in `dist/`.
- `npm run preview` serves the production build locally.
- `npm run lint` runs ESLint.

## Project structure

- `src/` contains the React application code.
- `public/` contains static assets served as-is.
- `vite.config.js` contains Vite configuration.
- `eslint.config.js` contains lint rules.
